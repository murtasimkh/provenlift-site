The Friction

1

"Book First Class" = 5 Steps

Click → Modal → Name → Email → Phone → Mindbody iframe → Then see schedule. Cold traffic bounces at the phone field.

2

ClassPass Books in 2 Clicks

Same studio, same inventory. ClassPass users skip the form entirely. Direct traffic penalized.

3

No "Guest Pass" Path

$50 intro offer exists but requires account creation first. No "hold a spot, pay at studio" option for walk-ins.

4

Zero Post-Opt-In Nurture

Email captured → added to generic newsletter. No sequence for "viewed schedule, didn't book" or "booked, didn't show."

The Fix (Logic Map Preview)

✓ Replace modal with inline calendar (public read-only)
✓ Add "Reserve Spot → Pay at Studio" button (no phone required)
✓ Webhook: Form submit → n8n → SMS "Spot held 15 min" + Email with class prep
✓ Branch: No-show → 30min SMS "Life happens. Next class on us?" + Deep link to rebook
✓ Branch: Trial used → Day 3 "How'd it feel?" → Day 7 Founder offer ($99/mo lock-in)
✓ UTM preservation across Mindbody → ClassPass → CRM for true CAC
